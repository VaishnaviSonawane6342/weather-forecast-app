
🌦️ Weather Forecast App

A modern and responsive Weather Forecast Web Application built using HTML, CSS, JavaScript, and OpenWeather API.
This application allows users to search for any city and get real-time weather information with a clean, attractive, and user-friendly interface.e.

🚀 Features

1.Search weather by city name
2.Real-time weather updates
3.Dynamic weather-based backgrounds
4.Temperature information
5.Feels Like temperature
6.Humidity details
7.Wind Speed
8.Sunrise & Sunset timings
9.Responsive Design

🛠️ Technologies Used
.HTML5
.CSS3
.JavaScript (ES6)
.OpenWeather API
.Bootstrap Icons
